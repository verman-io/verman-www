Version Manager (verMan)

  • nvm for Node.js;
  • rustup for Rust;
  • pyenv for Python.

    • MariaDB
    • Redis Valkey
  • Application servers
  • Reverse proxies and HTTP servers
  • Cluster management solutions
    • Kubernetes

  • …
